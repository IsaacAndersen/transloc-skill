# transloc-skill
Alexa skill for Duke Bus times.

To run locally:

Launch the Flask server: 
python server.py

To start a HTTP tunnel on port 5000, run this next:
./ngrok http 5000

You'll see some data, use the given https address as your endpoint in the Alexa skill dashboard.
